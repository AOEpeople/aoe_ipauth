Users manual
============

To add ip authentication to a website users or groups, go to your selected user/group and edit it. With the extension installed you will see additional "Auto-Login Group when coming from following IP/range" in the "Option"-tab.

Here you can then add IP/range in following form:
	- Simple IP: 192.168.2.4
	- Wildcard IP: 192.168.2.*
	- IP Range: 192.168.2.4-192.168.2.10
	- CIDR Range: 192.168.2.0/24
